# SQL_introduction

This project covers the fundamentals of SQL and MySQL: creating and
dropping databases and tables, inserting and querying data, updating
and deleting records, and aggregate functions like `COUNT`, `AVG`, and
`GROUP BY`.

## Learning Objectives

- What's a database, and the difference between SQL and NoSQL
- What RDBMS stands for
- How to create a database in MySQL
- What's a `PRIMARY KEY`
- How to `SELECT` data from a database, with conditions and ordering
- How to `INSERT`, `UPDATE`, and `DELETE` data in a database
- What's a subquery, and aggregate functions like `AVG`, `COUNT`, and
  `GROUP BY`
