# python-input_output

This project covers file I/O in Python: reading and writing text files,
JSON serialization/deserialization (both to strings and to files), and
converting between Python objects and JSON-compatible dictionaries. It
also revisits a classic technical-interview problem, Pascal's Triangle.

## Learning Objectives

- Why Python programming is awesome
- How to open a file and why you should use `with`
- How to write text in a file
- How to read the full content of a file
- How to read a file line by line
- What's JSON, and how to convert a Python data structure to/from it
- What is serialization/deserialization
- How to convert a JSON string to a Python data structure and vice versa

## Files

| File | Description |
|------|-------------|
| `0-read_file.py` | Reads and prints a text file |
| `1-write_file.py` | Writes a string to a text file |
| `2-append_write.py` | Appends a string to a text file |
| `3-to_json_string.py` | Converts an object to a JSON string |
| `4-from_json_string.py` | Converts a JSON string to an object |
| `5-save_to_json_file.py` | Saves an object to a JSON file |
| `6-load_from_json_file.py` | Loads an object from a JSON file |
| `7-add_item.py` | Adds CLI arguments to a list, saved as JSON |
| `8-class_to_json.py` | Builds a serializable dict from an object |
| `9-student.py` | `Student` class with `to_json()` |
| `10-student.py` | `Student` with a filterable `to_json(attrs)` |
| `11-student.py` | `Student` with `to_json()` and `reload_from_json()` |
| `12-pascal_triangle.py` | Builds Pascal's Triangle |
