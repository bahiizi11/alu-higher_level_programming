# python-data_structures

This directory is part of the `alu-higher_level_programming` repository and covers
basic Python data structures: lists, tuples, and simple algorithms built on top of
them without relying on built-in helper functions or imported modules.

## Learning Objectives

By the end of this project, you should be able to explain:

* What are lists and how to use them
* What are differences and similarities between lists and strings
* What are the most common methods of lists and how to use them
* How to use lists as stacks and queues
* What is a list comprehension and how to use them
* What are tuples and how to use them
* What is the difference between lists and tuples
* When to use tuples instead of lists and vice-versa
* What is a sequence
* What are mutable and immutable objects
* How does Python pass variables to functions

## Requirements

* All scripts run on Ubuntu 20.04 LTS using Python 3
* All files end with a new line
* The first line of all files is exactly `#!/usr/bin/python3`
* Code follows the pycodestyle style guide (version 2.x)
* All files are executable (`chmod +x`)
* No module is imported unless explicitly allowed by the task
* All functions and modules are documented

## Files

| File | Description |
| --- | --- |
| `0-print_list_integer.py` | Prints all integers of a list, one per line |
| `1-element_at.py` | Retrieves an element from a list by index, safely |
| `2-replace_in_list.py` | Replaces an element of a list at a specific position |
| `3-print_reversed_list_integer.py` | Prints all integers of a list in reverse order |
| `4-new_in_list.py` | Replaces an element in a list without modifying the original |
| `5-no_c.py` | Removes all `c` and `C` characters from a string |
| `6-print_matrix_integer.py` | Prints a matrix of integers |
| `7-add_tuple.py` | Adds two tuples together, element-wise |
| `8-multiple_returns.py` | Returns the length of a string and its first character |
| `9-max_integer.py` | Finds the biggest integer in a list |
| `10-divisible_by_2.py` | Returns a list of booleans for multiples of 2 |
| `11-delete_at.py` | Deletes the item at a specific position in a list |
| `12-switch.py` | Switches the values of two variables in 5 lines |

## Usage

Each exercise can be tested using its corresponding `N-main.py` test file:
