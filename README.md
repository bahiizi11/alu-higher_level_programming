# ALU Higher-Level Programming

This repository contains projects completed as part of the ALX/ALU Higher-Level
Programming curriculum. Each directory corresponds to a specific project
focused on Python fundamentals and best practices.

## Projects

### python-import_modules

This project covers how to import and use modules, functions, and variables
across different Python files. Topics covered include:

- Importing a function from another file
- Importing multiple functions from a module
- Using `sys.argv` to handle command-line arguments
- Reading names defined inside a compiled `.pyc` module
- Importing a variable from another file

| File | Description |
|------|-------------|
| `add_0.py` | Defines the `add` function used by `0-add.py` |
| `0-add.py` | Imports `add` from `add_0` and prints `1 + 2 = 3` |
| `calculator_1.py` | Defines `add`, `sub`, `mul`, and `div` functions |
| `1-calculation.py` | Imports and uses all four functions from `calculator_1` |
| `2-args.py` | Prints the number of and list of arguments passed to it |
| `3-infinite_add.py` | Prints the sum of all arguments passed to it |
| `4-hidden_discovery.py` | Prints all public names defined in `hidden_4.pyc` |
| `variable_load_5.py` | Defines the variable `a` used by `5-variable_load.py` |
| `5-variable_load.py` | Imports and prints the variable `a` from `variable_load_5` |
