# python-network_1

This project introduces HTTP networking in Python using both **urllib** and
**requests**.

## Learning Objectives

- Send HTTP GET and POST requests
- Read response headers
- Handle HTTP errors
- Send form data
- Parse JSON responses
- Use the GitHub REST API
- Authenticate with a Personal Access Token

## Files

| File | Description |
|------|-------------|
| 0-hbtn_status.py | Fetch status using urllib |
| 1-hbtn_header.py | Display X-Request-Id header using urllib |
| 2-post_email.py | POST email using urllib |
| 3-error_code.py | Handle HTTP errors using urllib |
| 4-hbtn_status.py | Fetch status using requests |
| 5-hbtn_header.py | Display X-Request-Id header using requests |
| 6-post_email.py | POST email using requests |
| 7-error_code.py | Handle HTTP errors using requests |
| 8-json_api.py | Search API and parse JSON |
| 10-my_github.py | Display GitHub user ID |
